EIGRP Routing Configuration


EIGRP routing protocol for IT/Main Router:

Router>enable
Router#
Router#configure terminal
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#interface Serial2/0
Router(config-if)#exit
Router(config)# router eigrp 1
Router(config-router)# network 201.157.30.0 255.255.255.0
Router(config-router)# network 199.151.1.0 255.255.255.0
Router(config-router)# network 199.151.2.0 255.255.255.0
Router(config-router)# network 199.151.3.0 255.255.255.0
Router(config-router)# network 199.151.4.0 255.255.255.0
Router(config-router)# network 199.151.5.0 255.255.255.0
