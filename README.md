# Hotel-Network-Management-System

The Hotel Network Management System is designed to provide seamless communication, efficient data transmission, and secure network operations across floors, each housing a different department of a hotel building. A star-like topology was designed, where each department’s router connected directly to the central IT router, with serial links enabling inter-router communication. Several essential servers were deployed, including DHCP for automatic IP assignment, DNS for domain name resolution, Web for online services, FTP for file transfers, and Email for communication. The project demonstrated expertise in subnetting, routing, and network services, ensuring seamless connectivity and operations.


# 1.1 What is a Cisco Packet Tracer?

Cisco Packet Tracer is a powerful network simulation tool and learning platform developed by Cisco Systems. It's designed for networking professionals and students to practice, learn, and experiment with various networking concepts and configurations.

| Network Simulation Tool | Versatile Capabilities | Collaborative Learning |
|---|---|---|
| Cisco Packet Tracer is an advanced network simulation and emulation software that allows students, instructors, and network administrators to design, simulate, and troubleshoot complex networks. | With far-reaching capabilities that include virtual hardware, protocols, and features, the Cisco Packet Tracer is used for network design, testing, gaming, and app development. | Cisco Packet Tracer promotes interactive, collaborative learning experiences with its easy-to-use interface and effective tutorials. |

# 1.2 Key Features:

## 1. Integrated Virtual Environment

Supports virtual environments to test various network setups, troubleshoot issues or complications, and understand complex concepts.

## 2. Multi-Protocol Object-Oriented Environment

It supports multiple protocols and allows users to create and visualize complex network scenarios.

## 3. Resourceful Network Elements Library

The software offers a wide range of network elements, such as end devices, peripherals, and cables, which allow for easy design and planning of complex networks.
